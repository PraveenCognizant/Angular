Directives

1. @if / @else (Structural — shows or removes from DOM)

@if (isLoggedIn) {

Welcome! You are logged in.

} @else {

Please log in to continue.

}

Score: {{ score }}  

@if (score >= 90) { Grade A } @else if (score >= 70) { Grade B } @else if (score >= 50) { Grade C } @else { Grade F }

2. @for (Structural — loops over arrays)

track is required — it helps Angular efficiently update only changed items.

Count: {{ fruits.length }}
    @for (fruit of fruits; track fruit; let i = $index; let isEven = $even) {
  • {{ i + 1 }}. {{ fruit }}
    • } @empty {
  • No fruits! Click "Add Fruit" above.
    • }

3. @switch / @case (Structural — like JavaScript switch)

@switch (status) { @case ('active') {
🟢 Status: Active
} @case ('inactive') {
⚫ Status: Inactive
} @case ('pending') {
🟡 Status: Pending
} @default {
❓ Unknown status
} }

4. ngClass & ngStyle (Attribute Directives)

These don't add/remove elements — they change the appearance of existing elements.

This box changes style dynamically with ngClass + ngStyle!
Font size: {{ fontSize }}px | Highlighted: {{ isHighlighted }}
