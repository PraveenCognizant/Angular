Signals (Angular 16+)

Signal API Quick Reference: 
const x = signal(0);         // create signal with initial value
x()                           // READ the current value (call it like a function)
x.set(5)                      // WRITE: replace with new value
x.update(n => n + 1)          // WRITE: transform current value
computed(() => x() * 2)       // DERIVED signal (auto-updates)
effect(() => console.log(x())) // SIDE EFFECT (runs when deps change)
