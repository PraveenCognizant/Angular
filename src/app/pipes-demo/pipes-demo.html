Pipes

Syntax: {{ '{{' }} value | pipeName : argument {{ '}}' }}

Text Pipes

Pipe Result
| uppercase {{ name | uppercase }}
| lowercase {{ name | lowercase }}
| titlecase {{ name | titlecase }}
| slice:0:7 {{ name | slice:0:7 }}
| reverse (custom) {{ textToReverse | reverse }}

Number & Date Pipes

Pipe Result
| currency {{ price | currency }}
| currency:'INR' {{ price | currency:'INR' }}
| number:'1.2-4' {{ pi | number:'1.2-4' }}
| percent {{ ratio | percent }}
| date:'short' {{ today | date:'short' }}
| date:'dd/MM/yyyy' {{ today | date:'dd/MM/yyyy' }}

JSON Pipe (great for debugging!)

Converts any object to formatted JSON string.

{{ person | json }}

Chaining Pipes

Pipes can be chained — the output of one becomes input of the next.

name | uppercase | reverse
→ {{ name | uppercase | reverse }}

today | date:'fullDate' | uppercase
→ {{ today | date:'fullDate' | uppercase }}


Reversed: {{ customText | reverse }}
