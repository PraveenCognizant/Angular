Data Binding (Most Asked in Interviews!)

1. Interpolation — {{ '{{' }} expression {{ '}}' }}

Evaluates TypeScript expressions and renders them as text. One-way: class → template.

Variable: {{ userName }}

Math: {{ 10 + 5 * 2 }}

Method call: {{ userName.toUpperCase() }}

Ternary: {{ clickCount > 0 ? 'Clicked!' : 'Not yet clicked' }}

2. Property Binding — [property]="expression"

Sets an HTML element's property dynamically. One-way: class → template.

Square brackets [ ] tell Angular: "evaluate the right side as TypeScript".

[src] example:

Background set via [style.background]="{{ highlightColor }}"

3. Event Binding — (event)="handler()"

Listens to DOM events and calls TypeScript methods. One-way: template → class.

Parentheses ( ) tell Angular: "call the method when this event fires".

Use $event to pass the native event object to the handler.

Last key: {{ lastKey }}
Move your mouse here!
X: {{ mouseX }}   Y: {{ mouseY }}

4. Two-Way Binding — [(ngModel)]="property"

Data flows in both directions: class ↔ template simultaneously.

[(ngModel)] is a shorthand for [ngModel]="val" (ngModelChange)="val=$event"

Requires FormsModule in imports array!
