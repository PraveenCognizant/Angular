Lifecycle Cheat Sheet (Interview Quick Reference)

Hook When it fires Best used for
constructor() Class is instantiated Dependency injection only
ngOnChanges() @Input() value changes React to parent input updates
ngOnInit() Once after first render API calls, data loading ⭐
ngAfterViewInit() Template + children rendered @ViewChild DOM access
ngOnDestroy() Before component removal Unsubscribe, cleanup ⭐
